## Latest Experiment Readout

Panda to tiger object replacement:

- `base_only` preserves the source layout but softens the panda.
- `direct_target` and `anchor_only` create a strong tiger-face signal, but the
  body remains panda-like. This is the current overlay/hybrid failure mode.
- `decoupled_rec` activates the reconstruction branch and keeps the structure
  somewhat more stable, but it does not yet produce full object replacement.
- Branch diagnostics in `scripts/run_panda_tiger_diagnostics.sh` show that
  flow-only and clean-anchor edits are the strongest, region-anchor weakens the
  edit without solving the hybrid structure, trajectory preservation further
  protects the panda body, and feature-only target/source branches are too weak
  to drive replacement by themselves.
- The current bottleneck is object-level rewrite, not only tuning the
  reconstruction/edit strength ratio.
- Rewrite probes in `scripts/run_panda_tiger_rewrite_probe.sh` test whether
  adding initial latent noise inside the edit mask can free the source object
  from the inverted panda trajectory. Mild subject noise changes the body
  slightly but still gives a hybrid, stronger subject noise destroys face
  coherence, core-only noise keeps the original failure mode, and adding anchor
  with noise strengthens the tiger face without rewriting the full body.
- This suggests the current RF edit field needs an object-level generation /
  replacement mechanism, not only more freedom inside the same attention mask.

Panda sunglasses local insertion:

- `sunglasses_local` remains the best current SD3 local result.
- external-mask diagnostics show that spatial support matters, but the current
  external proposal mask is not reliable enough to replace attention-derived
  support.
- simple eye-box intersection protects the lower face but moves the glasses
  upward, so it is useful as a diagnostic rather than a default.
- Cross-method comparison with RF-Solver and FireFlow shows that those methods
  place smaller glasses closer to the original eye locations while changing far
  fewer pixels. Their extracted proposal-source masks are much smaller than the
  current attention subject mask.
- Using only the high-confidence core as the ODE edit support is too narrow and
  does not generate glasses. Using subject support for generation but core
  support for final latent blending preserves the face but cuts the edit into a
  local patch artifact. This suggests the useful idea is not simply "smaller
  mask", but separating broad contextual generation from a precise local
  structural/attention injection mechanism.
